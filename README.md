# Robusta

Human + Web AI (Tori) + Code AI (Komi) — three-way collaboration product, BYOK.

## Stack
- Next.js 15 (App Router, `output: 'export'`)
- React 19, TypeScript strict
- Tailwind CSS, Zustand, Dexie (IndexedDB)

## Scripts
- `npm run dev` — local dev server
- `npm run build` — static export to `out/`
- `npm run typecheck` — `tsc --noEmit`

## Source of truth
- Specs: Confluence `Robusta` space, `Task_YYYY-MM-DD` pages
- Design base: https://blend.ai4min.com/design1/ko (Robusta inherits 80%, yellow canvas differentiates)
- Production URL: https://robusta.ai4min.com
